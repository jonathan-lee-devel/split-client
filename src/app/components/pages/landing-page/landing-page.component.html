What can this application do?

Split is an application for people sharing accommodation who want to track their expenses.

Current Features

  • Creating a property to manage
  • Invite other tenants via e-mail
  • Create expenses
  • Create expense distribution assignments
  • View monthly expense reports

    • Upcoming Features

    Split is currently in a pre-production phase and more features are always being added

    Some Upcoming Features
  • Monthly expense archives
  • Repeating expenses
  • Tenant reliability scores
